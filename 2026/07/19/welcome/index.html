博客开张

发表于 2026-07-19 分类于 随笔

欢迎来到我的个人博客！

这是我用 Hexo 搭建、部署在 GitHub Pages 上的小站。这里会记录：

  • 技术：编程学习、项目实战、工具分享
  • 生活：阅读、旅行、日常感悟
  • 项目：自己做的一些小玩意儿

为什么建这个博客

一方面是记录自己的成长轨迹——很多知识学了不用很快就忘了，写下来是最好的巩固方式。另一方面也希望能与读到这里的你产生一些连接，哪怕只是一行代码的启发或者一段文字的共鸣。

博客会持续更新，欢迎常来看看。如果你有什么想聊的，可以在 GitHub 上找我。
